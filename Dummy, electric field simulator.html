Electric Field Playground

This web app simulate how electric field behave around charges. The field will spread out of the positive charge and spread in to the negative charge. The Field also change whereas the charge moved or if there are another charge inside the field. In this web app, the simulation not yet ready. The charge cannot moved using user mouse. The user also cannot add another charge the in the playground canvas. But there here in this mockup simulation, a charge cirle and some field already drawn but still not interactive for playground web app.

The input and the output

  • The web app output are a canvas. A canvas whereas there are an electric field and charges that interactive with each other.
  • In future, maybe if there is more time, an E field and distace r graph can be drawn too with another canvas or graph. Either for the E field, or Coulomb force.
  • Therefore, for the input of this web app, its only charge value. But the user can and multiple charge here. Positive or negative.

How it words actually?

  • This is just using simple equation for E field and Coulomb force from undergraduate physics Electromagnetism.
  • It just an iteration of the distace from r and r' where the user drag the charge. Same for the Coulomb force. The differences is the function.
  • There has to be two function at leasst to draw the arrow field for both E field and force arrow.
